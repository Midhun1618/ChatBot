# AI Chatbot using LangChain & OpenAI

## Overview
This project demonstrates a Python-based AI chatbot built using LangChain and OpenAI's LLMs. It showcases prompt engineering, conversational AI workflows, and basic agent-style orchestration.

## Technologies Used
- Python
- LangChain
- OpenAI API
- Prompt Engineering

## Features
- Interactive conversational chatbot
- Modular and scalable design
- Controlled AI responses using prompts

## How to Run
1. Add your OpenAI API key in `.env`
2. Install dependencies:
   pip install langchain openai python-dotenv
3. Run:
   python terminal-chatbot.py
